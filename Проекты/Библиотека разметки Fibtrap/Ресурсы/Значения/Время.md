---
Prefix: time
---

| Ключ      | Значение | Описание       |
| --------- | -------- | -------------- |
| none      | 0s       | без задержки   |
| very-fast | .1s      | очень быстро   |
| fast      | .2s      | быстро         |
| norm      | .3s      | нормально      |
| slow      | .5s      | медленно       |
| very-slow | .8s      | очень медленно |

```css
    --time-none: 0s;
    --time-very-fast: .1s;
    --time-fast: .2s;
    --time-norm: .3s;
    --time-slow: .5s;
    --time-very-slow: .8s;
```
