## Общие

| Ключ | Описание                                            |
| ---- | --------------------------------------------------- |
| px   | пиксели (абсолютная единица)                        |
| vh   | проценты от **высоты окна** (`100vh = 100% высоты`) |
| vw   | проценты от **ширины окна** (`100vw = ширина`)      |
| vmax | меньшая из `vh` или `vw`                            |
| vmin | большая из `vh` или `vw`                            |
| %    | проценты от родительского элемента                  |
## Текстовые

| Имя | Описание                                      |
| --- | --------------------------------------------- |
| em  | относительная к **размеру шрифта родителя**   |
| rem | относительная к **корневому шрифту** (`html`) |
| ex  | высота строчной буквы `x` в текущем шрифте    |
| ch  | ширина символа `0` в текущем шрифте           |
## Мировые

| Имя | Описание                    |
| --- | --------------------------- |
| cm  | сантиметры                  |
| mm  | миллиметры                  |
| in  | дюймы (1in = 2.54см = 96px) |
| pt  | пункты (1pt = 1/72 дюйма)   |
## Другое

| Имя | Описание                    |
| --- | --------------------------- |
| pc  | пики (1pc = 12pt)           |
| fr  | единица доли в Grid layout  |
