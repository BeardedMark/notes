# Таблица всех локаций
Мировые области которые можно посещать

| name        | type    | default | unique | desctiption  |
| ----------- | ------- | ------- | ------ | ------------ |
| x           | ?int    |         |        | Координата X |
| y           | ?int    |         |        | Координата Y |
| level       | int     | 1       |        | Уровень      |
| size        | int     | 1       |        | Размер       |
| is_peaceful | boolean | false   |        | Мирная ли    |
| tags        |         |         |        | Теги         |
- x : y - задают мировое расположение локации
- level - контролирует генерацию контента
- size - влияет на кол-во контента 
- is_peaceful - является ли локация мирной
	- нельзя западать на игроков
	- мобы не нападают на игроков
