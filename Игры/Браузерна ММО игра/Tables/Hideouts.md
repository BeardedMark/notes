# Таблица всех убежищ
Области игроков созданные на локации

| name          | type                             | default | unique | desctiption |
| ------------- | -------------------------------- | ------- | ------ | ----------- |
| location_code | [[Location]]                     |         |        |             |
| user_id       | [[User]]                         |         |        |             |
| level         | int                              | 1       |        | Уровень     |
| inventory     | ?array [[Item Instance]]         |         |        | Тайник      |
| enemies       | ?array [[Enemy Instance]]        |         |        | Охрана      |
| constructions | ?array [[Construction Instance]] |         |        | Конструкции |

- Во время нахождения на локации есть возможность построить убежище
- Для постройки требуются ресурсы и соблюдение условий
- Начальное убежище имеет 1 уровень
- Уровень влияет на объем убежища
- Объем убежища позволяет размещать содержимое

## Осада
- Создается открытый бой за убежище (рейд)
- Владелец или кто угодно может принять бой (антирейд)
- При победе у убежища снимается уровень
- Кол-во снятых уровней зависит от предметов штурма
- При снятии уровня выпадают предметы или нападает охрана
