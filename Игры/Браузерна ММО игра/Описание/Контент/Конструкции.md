Не перемещаемые объекты, которые содержат внутри себя предметы
Основные игровые элементы для взаимодействия
#### Параметры
- Группа - определяет суть предмета
- Категория - определяет основу предмета
- Теги - дополняет предмет деталями
- Минимальный уровень - от какого уровня может быть найден
- Максимальный уровень  - до какого уровня может быть найден
- Шанс выпадения - базовое значение шанса выпадения
- [[Биомы]] - где встречается  базовым шансом
- [[Локации]] - где встречается базовым шансом
- [[Модификаторы]] - список возможных свойств

- Содержимое - рецепт который описывает предметы в предмете
	- Предмет - что участвует в рецепте
	- Кол-во - сколько нужно предметов
	- Шанс - вероятность выпадения при разборе
- Собираемое - можно создать из других предметов
	- Условия для сборки
- Разбираемое - можно извлечь предметы
	- Условия для разбора предмета
- Действия - что произойдет при активации предмета
	- Условия для активации предмета
#### Экземпляр
- Объект
- Уровень
- Модификаторы
#### Действия
- Подбор - переместить предмет в инвентарь
- Выбросить - переместить предмет из инвентаря в другое место
- Экипировать - переместить предмет из инвентаря в слот экипировки
- Собрать - создание нового предмета из других
- Разобрать - уничтожить предмет с шансом достав из него предметы
