Созданные пользователями игровые персонажи
#### Параметры
- Опыт - определяет общее значение игрового прогресса
- Инвентарь - предметы у персонажа на руках
- Экипировка - надетые предметы на персонажа
- Характеристики - сила, ловкость, интеллект
- Таланты - фиксированные модификаторы дерева умений
- Достижения - счетчики для определения пассивной прокачки
	- убито мобов
	- побеждено в боях
	- проиграно боев
	- боев в ничью
	- собрано объектов
	- разобрано объектов
	- завершено заданий
#### Методы
- Уровень - высчитывается по формуле исходя из опыта
#### Зависимости
- [[Поединки]] - где персонаж принимал участие
	- Завершенные
	- Активные
