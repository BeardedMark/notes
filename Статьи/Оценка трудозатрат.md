# Эффективное планирование задач через оценку сложности
Статья раскрывает методику расчета времени на выполнение задач исходя из их сложности. Вы узнаете, как оценивать трудозатраты, корректировать план работы и определять, сколько задач можно взять в работу за доступное время. Примеры расчетов помогут рационально распределять ресурсы и избегать перегрузки команды

Дает результат при минимальном количестве данных
Основывается на теоретических данных
Помогает научится оценивать время по сложности задачи
## Время на задачу по сложности
Вычисление времени в часах на задачу по всем сложностям исходя из выделенного или затраченного времени на все задачи

Дано:
1. Общее количество часов работы
2. Список готовых задач
	- Сложность задачи

Решение:
1. Доля сложности задачи во всей работе
	- Сложность задачи / Сумма всех уровней сложности
2. Часов на задачу
	- Доля сложности задачи во всей работе * Общее количество часов работы
## Возможные задачи по сложности
Решение:
1. Количество задач по сложности
	- Выделенное время / Время на задачу по сложности
## Сумма времени на все задачи
Получение времени в часах, которое обозначает свободное или преизбыточное время на выполнение запланированных задач

Дано:
1. Список задач в планах
	- Сложность задачи

Решение:
1. Количество задач каждой сложности
2. Время выполнения всех задач по сложности
	- Количество задач каждой сложности * Время на задачу по сложности
3. Сумма времени на все задачи

Ответ:
1. Положительное число - свободное время
2. Отрицательное число - переизбыток задач 
## Планирование задач
Дано:
1. Выделенное время

Решение:
1. Доступное время
	- Выделенное время - Сумма времени на все задачи
2. Объем задач
	- Доступное время / Время на задачу по сложности

Ответ:
1. Положительное число - сколько задач можно взять в работу
2. Отрицательное число - сколько задач нужно убрать из плана
