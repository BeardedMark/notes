### Начало

1. Скачивание директории `git clone` `UrlProject`
2. Создание проекта `composer create-project laravel/laravel Laravel`
3. Запуск сервера `php artisan` `serve`

### Разработка

1. Создать модель с миграцией `php artisan` `make:model` `NameModel` `-m`
2. Создать контроллер к модели `php artisan` `make:controller` `NameController` `--resource` `--model=Chat`

### Миграции

1. Запустить миграцию `php artisan` `migrate`
2. Откатить миграцию `php artisan` `migrate:rollback`
3. Удалить миграцию `php artisan` `migrate:reset`
4. Обновить миграцию `php artisan` `migrate:fresh` `--seed`

### Сидеры

1. Создать сидер для заполнения данных `php artisan` `make:seeder` `NameSeeder`
2. Запустить сидер `php artisan` `db:seed`

### Разное

1. Создать символьную ссылку `php artisan` `storage:link`
2. Очистить кэш роутов `php artisan` `route:clear`
