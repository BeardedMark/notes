### Начало

```bash
# Создание проекта
composer create-project laravel/laravel Laravel
```

```bash
# Переход в директорию
cd Laravel
```

```bash
# Запуск сервера
php artisan serve
```

### Разработка

```bash
# Создать модель с миграцией
php artisan make:model Name -m
```

```bash
# Создать контроллер к модели
php artisan make:controller NameController --resource --model=Name
```

### Миграции

```bash
# Запустить миграцию
php artisan migrate
```

```bash
# Откатить миграцию
php artisan migrate:rollback
```

```bash
# Удалить миграции
php artisan migrate:reset
```

```bash
# Обновить миграции
php artisan migrate:fresh
```

### Сидеры

```bash
# Создать сидер
php artisan make:seeder NameSeeder
```

```bash
# Запустить сидер
php artisan db:seed
```

### Разное

```bash
# Создать символьную ссылку
php artisan storage:link
```

```bash
# Полная очистка кэша
php artisan optimize:clear
```
