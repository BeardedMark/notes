1. Установка Composer
    1. Скачать с [сайта](https://getcomposer.org/download/) файл по ссылке в тексте [Composer-Setup.exe](https://getcomposer.org/Composer-Setup.exe)
    2. Запустить файл `Composer-Setup.exe`
    3. Далее - далее - далее…
2. Установить PHP
    1. Скачать с [сайта](https://windows.php.net/download) версию **`VS16 x64 Thread Safe** (Zip)`
    2. Распаковать архив по пути `C:\\php`
    3. Создать файл `php.ini` на основе `php.ini-development`
    4. В файле изменить путь `extension_dir = "C:\\php\\ext"`
    5. В файле раскомментировать настройки:
        - `extension=curl`
        - `extension=fileinfo`
        - `extension=openssl`
        - `extension=pdo_mysql`
        - `extension=mbstring`
        - **`extension=gd` (composer require phpoffice/phpspreadsheet → composer require maatwebsite/excel)**
3. Установка Laravel
    1. В папке где будет проект открыть `cmd` терминал
    2. В терминале прописать команду `composer create-project laravel/laravel *name*`
