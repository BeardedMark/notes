# Директории серверной части

### Стандартные директории
| Название папки | Описание                        |
| -------------- | ------------------------------- |
| Http           |                                 |
| ↳ Controllers  | контроллеры                     |
| ↳ Middleware   | фильтры/промежуточная логика    |
| ↳ Requests     | классы валидации входных данных |
| Models         | модели Eloquent (сущности БД)   |
| Services       | бизнес-логика вне моделей       |
### Дополнительные директории

| Название папки | Описание                                                  |
| -------------- | --------------------------------------------------------- |
| Domain         | бизнес-домены (DDD-подход, микросервисы внутри монолита). |
| Application    | сервисы, DTO, use-cases (слой прикладной логики).         |
| Infrastructure | адаптеры, интеграции с API, репозитории                   |
| Presentation   | контроллеры, запросы, ресурсы                             |
| Traits         | общие методы для объектов                                 |
| Schemas        | DTO, контракты данных, API-ответы                         |
| Instances      | вспомогательные классы, адаптеры                          |
| Mail           | email-шаблоны                                             |
| Notifications  | уведомления (mail, slack, sms и т.д.)                     |
| Events         | события                                                   |
| Listeners      | обработчики событий                                       |
| Jobs           | фоновые задания (очереди)                                 |
| Policies       | политика доступа                                          |
| Providers      | сервис-провайдеры                                         |
| Factories      | фабрики для тестов/сидов                                  |
