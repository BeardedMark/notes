## Создание
1. Синхронизироваться с Git-репозиторием
2. Инициализировать проект `npm init -y`
## Обновление
1. Изменить параметры в `package.json`
2. Собрать пакет `npm run build`
3. Опубликовать пакет `npm publish --access public`
## Использование
1. По CDN `https://cdn.jsdelivr.net/npm/<name>@<version>/<directory>`
2. Подключение `npm install <package>@<ver>`
