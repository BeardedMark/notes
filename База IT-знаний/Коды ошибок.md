# Описание кодов ошибок

## Клиентские ошибки

| code | description            | note                                                       |
| ---- | ---------------------- | ---------------------------------------------------------- |
| 400  | Bad Request            | Неверный запрос: синтаксис или данные неправильные         |
| 401  | Unauthorized           | Требуется авторизация (например, отсутствует токен)        |
| 403  | Forbidden              | Доступ запрещён, даже если авторизован                     |
| 404  | Not Found              | Ресурс не найден                                           |
| 405  | Method Not Allowed     | Метод не разрешён для этого пути                           |
| 406  | Not Acceptable         | Сервер не может отдать запрошенный формат ответа           |
| 408  | Request Timeout        | Время ожидания запроса истекло                             |
| 409  | Conflict               | Конфликт запроса с текущим состоянием (например, дубликат) |
| 410  | Gone                   | Ресурс был удалён и больше недоступен                      |
| 413  | Payload Too Large      | Слишком большой размер тела запроса                        |
| 415  | Unsupported Media Type | Формат запроса не поддерживается сервером                  |
| 429  | Too Many Requests      | Слишком много запросов за короткий период (rate limit)     |
## Серверные ошибки

| code | description           | note                                                  |
| ---- | --------------------- | ----------------------------------------------------- |
| 500  | Internal Server Error | Внутренняя ошибка сервера                             |
| 501  | Not Implemented       | Метод не реализован на сервере                        |
| 502  | Bad Gateway           | Сервер получил плохой ответ от другого сервиса        |
| 503  | Service Unavailable   | Сервер временно недоступен (перегрузка, обслуживание) |
| 504  | Gateway Timeout       | Превышено время ожидания ответа от другого сервиса    |
