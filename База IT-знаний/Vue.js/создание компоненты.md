### Создание компонента

1. Создать файл в папке `src/components/ComponentName.vue`
2. Использовать структуру: `<template>`, `<script>`, `<style>`

### Базовый компонент

```vue
<template>
  <div class="component-name">
    <h2>{{ title }}</h2>
    <p>{{ description }}</p>
  </div>
</template>

<script setup>
defineProps({
  title: String,
  description: String
})
</script>

<style scoped>
.component-name {
  padding: 20px;
}
</style>
```

### Использование компонента

```vue
<template>
  <ComponentName 
    title="Заголовок" 
    description="Описание"
  />
</template>

<script setup>
import ComponentName from '@/components/ComponentName.vue'
</script>
```

### Options API (альтернативный синтаксис)

```vue
<template>
  <div>{{ message }}</div>
</template>

<script>
export default {
  name: 'ComponentName',
  data() {
    return {
      message: 'Привет!'
    }
  },
  methods: {
    handleClick() {
      this.message = 'Клик!'
    }
  }
}
</script>
```

### Composition API (современный синтаксис)

```vue
<template>
  <div>
    <p>{{ count }}</p>
    <button @click="increment">+</button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const count = ref(0)

const increment = () => {
  count.value++
}

const doubleCount = computed(() => count.value * 2)
</script>
```

### Props и Emits

```vue
<template>
  <div>
    <p>{{ title }}</p>
    <button @click="handleClick">Клик</button>
  </div>
</template>

<script setup>
defineProps({
  title: {
    type: String,
    required: true
  }
})

const emit = defineEmits(['click'])

const handleClick = () => {
  emit('click', 'данные')
}
</script>
```
