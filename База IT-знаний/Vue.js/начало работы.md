### Установка Vue.js

1. Установка через npm
   1. Создать проект `npm create vue@latest`
   2. Выбрать опции установки (TypeScript, Router, Pinia и т.д.)
   3. Перейти в папку проекта `cd <project-name>`
   4. Установить зависимости `npm install`
   5. Запустить dev сервер `npm run dev`

2. Установка через CDN
   ```html
   <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
   ```

### Создание проекта


```bash
npm create vue@latest my-project
cd my-project
npm install
npm run dev
```

### Структура проекта

- `src/` — исходный код
- `src/components/` — компоненты
- `src/views/` — страницы (при использовании Router)
- `src/router/` — настройки роутинга
- `src/store/` — хранилище состояния (Pinia)
- `public/` — статические файлы
- `package.json` — зависимости проекта

### Основные команды

```bash
npm run dev        # запуск dev сервера
npm run build      # сборка для production
npm run preview    # предпросмотр production сборки
```

### Базовый пример

```vue
<template>
  <div>
    <h1>{{ message }}</h1>
    <button @click="increment">Счет: {{ count }}</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const message = ref('Привет, Vue!')
const count = ref(0)

const increment = () => {
  count.value++
}
</script>
```
