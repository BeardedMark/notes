# Инструкция по созданию сайта на хостинге

1. [Создать сайт](https://hosting.timeweb.ru/sites) с привязкой к **директории** и **домену**
	1. Создать **директорию** в [Файловом менеджере](https://hosting.timeweb.ru/fileman) на хостинге
	2. [Купить домен](https://hosting.timeweb.ru/domains/registration) для доступа к сайту по **ссылке**
2. Подключить сертификат [SSL Let's Encrypt](https://hosting.timeweb.ru/ssl#order)  как **Физ. лицо**
3. [Создать базу](https://hosting.timeweb.ru/mysql) данных **MySQL**
	1. Добавить доступы по IP ([узнать свой IP](https://2ip.ru/))
