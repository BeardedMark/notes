### Вариант №1

1. Открыть настройки
2. Перейти в раздел Workbench
3. Перейти в подраздел Editor Management
4. Включить галочку Wrap Tabs

### Вариант №2

1. Нажать сочетание клавиш `ctrl + shift + p`
2. Выбрать пункт `Preferences: Open User Settings (JSON)`
3. Внести строку настройки в файл `settings.json`

```json
"workbench.editor.wrapTabs": true
```
