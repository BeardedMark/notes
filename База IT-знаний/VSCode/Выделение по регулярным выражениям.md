1. Установить расширение [Highlight](https://marketplace.visualstudio.com/items?itemName=fabiospampinato.vscode-highlight)
2. Перейти в настройки расширения
3. Ввести строки настройки в файл `settings.json`

```json
"highlight.regexes": {

"((?<!-)\\\\bcol\\\\b-*\\\\S*\\\\b)": {
            "color": "#ff80ff"
        },
        "((?<!-)\\\\brow\\\\b(?!-))": {
            "color": "#80ff80"
        },
        "((?<!-)\\\\bcontainer\\\\b(?!-))": {
            "color": "#ffff80"
        },
        "((?<!-)\\\\bfib\\\\b-*\\\\S*\\\\b)": {
            "color": "#7B61FF"
        }
    }
```
