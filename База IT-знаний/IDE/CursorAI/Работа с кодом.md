### Выделение кода
1. Выделить код → `Ctrl+L` — чат с контекстом
2. Выделить код → `Ctrl+K` — встроенное редактирование
3. Множественное выделение → `Ctrl+D` — выделить следующее вхождение

### Редактирование
1. Встроенное: `Ctrl+K` → описать изменения
2. Через чат: `Ctrl+L` → запросить изменения → применить
3. Компиляция: `Ctrl+I` → создать новый файл

### Контекст
- Открытые файлы — автоматически в контексте
- Выделенный код — приоритетный контекст
- `.cursorrules` — правила проекта
