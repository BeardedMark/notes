### Открытие чата
- `Ctrl+L` / `Cmd+L` — открыть чат
- `Ctrl+K` / `Cmd+K` — встроенное редактирование
- `Ctrl+I` / `Cmd+I` — компиляция в новом файле

### Навигация
- `Ctrl+Shift+L` — открыть чат с выделенным кодом
- `Ctrl+Shift+K` — встроенное редактирование выделенного
- `Esc` — закрыть чат/панель

### Редактирование
- `Tab` — принять предложение
- `Ctrl+→` / `Cmd+→` — принять слово
- `Ctrl+Shift+→` / `Cmd+Shift+→` — принять строку
- `Alt+[` / `Alt+]` — предыдущее/следующее предложение

### Работа с кодом
- `Ctrl+/` / `Cmd+/` — комментарий/объяснение кода
- `Ctrl+Shift+P` — палитра команд
