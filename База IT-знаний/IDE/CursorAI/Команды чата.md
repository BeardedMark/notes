### Базовые команды
- `/fix` — исправить ошибки в коде
- `/explain` — объяснить код
- `/test` — создать тесты
- `/doc` — добавить документацию

### Редактирование
- `/edit` — редактировать код
- `/refactor` — рефакторинг
- `/optimize` — оптимизировать код

### Создание
- `/create` — создать файл/компонент
- `/generate` — сгенерировать код

### Анализ
- `/review` — код-ревью
- `/debug` — отладка
- `/analyze` — анализ кода
